## Installation

### Requirements
* [Required] - [Boost]
* [Required MacOSX] - [restclient-cpp]

Installing dependencies on MacOsX, you need brew

```bash
$ brew tap mrtazz/oss
$ brew install restclient-cpp
$ brew install boost
```

For Linux you only need boost

for ArchLinux
```bash
$ pacman -S boost
```

for Ubuntu
```bash
apt-get install libboost-all-dev
```
